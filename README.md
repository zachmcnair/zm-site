# Zvpply

Personal portfolio and professional site for Zach McNair.

## Features

- Modern, clean design with Faktum font family
- Last.fm scrobbler integration
- Responsive layout
- Dark mode support
- Optimized for SEO

## Tech Stack

- Next.js 15
- TypeScript
- Tailwind CSS
- Faktum font family
- Vercel Analytics

## Development

```bash
npm run dev
```

## Deployment

Deployed on Vercel at [zvpply.com](https://zvpply.com)
